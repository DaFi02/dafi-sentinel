# DAFI Sentinel

DAFI Sentinel is a security-first incident investigation workbench.
PR1 is only Python metadata, pytest, contracts, and guard tests.
```bash
uv sync
uv run pytest
```
## Later slices

PostgreSQL + pgvector, Podman infra, FastAPI, React + TypeScript + Vite,
auth implementation, LangGraph, and scikit-learn arrive later.
Grafana and Prometheus are out of scope for this product.
